jquery 超级select插件 v4.0.0.0版本

支持汉字、头写拼音、英文快速定位查询的超级select插件。可方向键、tab 键快速选择。

此插件能处理适合于 改变html 标签的input和select的功能,使input、select操作更加方面。使得select也可以进行填写功能。

支持 ie6.0、7.0、8.0、firefox all 版本。

js code:

第一种调用方法input 与select 的功能并存

$(document).ready(function(){
$('#sssss').selectseach();
});

第二种调用方法只有select 功能

$(document).ready(function(){
$('#sssss').selectseach({input:false});
});
